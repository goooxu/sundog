# sundog 渲染技术报告

这是一份**面向没有计算机图形学背景的工程师**的完整技术报告：从"数字图像是什么"
一路讲到"RTX 5090 上每秒数十亿次求交是怎么发生的"。数学假设读者具备
微积分、线性代数与概率论基础；每个图形学概念都先给直觉与图，再给严格推导，
最后落到 sundog 源码的具体函数。

所有渲染对比图可由 `scripts/render-report-figures.sh` 与
`scripts/gen-report-charts.py` 再生。

## 阅读路径

- **完整阅读**（推荐，约 3-4 万字）：按 01 → 17 → 附录顺序。
- **只关心原理**：01 → 02 → 03 → 04 → 05（成像、渲染方程、蒙特卡洛、路径追踪、材质），可加 13 章 13.1–13.2（辐射传输）、14 章（水面三层组合）、15 章（环境光重要性采样——第 3 章逆变换采样的实战应用）、16 章（透明阴影与嵌套介质）与 17 章（微表面透射——第 5 章 GGX 从反射到透射的收官）。
- **只关心 GPU 工程**：01（速览）→ 09 章 9.1（OptiX 应用七步流程总览）→ 08 → 09 → 10 → 11 → 12 → 13 → 14 → 15 → 16 → 17（加速结构、OptiX、决定性、验证、物理装载、体积渲染、水面、环境光照、透明阴影与嵌套介质、粗糙电介质）。
- **想看"常见陷阱"**：直接读[附录](appendix-pitfalls.md)——朴素实现最容易犯的六类计算错误。

## 延伸阅读

- Pharr, Jakob, Humphreys, *Physically Based Rendering: From Theory to Implementation*（PBRT，本领域标准教材）
- Shirley 等，*Ray Tracing in One Weekend* 系列（入门路径追踪的经典教程；附录剖析的若干朴素写法即源自其早期版本）
- NVIDIA, *OptiX Programming Guide*（本项目工程层的官方参考）
- Veach 的博士论文（MIS 的出处）
